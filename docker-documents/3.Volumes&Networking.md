### â“ Why Use Volumes?

* Containers are **ephemeral** (temporary).
* Data **inside containers is lost** when the container is deleted.
* Volumes allow data to **persist** even after the container is removed.

### ğŸ”¹ Network Drivers

| Driver    | Description                                         |
